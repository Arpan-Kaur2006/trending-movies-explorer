
# 🎬Trending Movies Explorer

## 📌 Project Overview

Trending Movies Explorer is a web application that allows users to discover currently trending movies using real-time data from the TMDB API. The app provides an interactive interface where users can search, filter, and sort movies based on different criteria.

This project focuses on building a responsive UI and applying core JavaScript concepts such as API handling, DOM manipulation, and array methods.

---

##  Features

*  **Trending Movies Display**
  View a list of currently trending movies with posters, ratings, and release dates.

* **Search Functionality**
  Search movies by title in real time.

* **Filtering**
  Filter movies based on rating (e.g., above 7).

* 🔃 **Sorting**
  Sort movies by:

  * Rating
  * Popularity

*  **Favorites (Optional)**
  Save movies using localStorage.

*  **Dark Mode (Optional)**
  Toggle between light and dark themes.

---

##  API Used

* **TMDB (The Movie Database API)**
  Provides movie data including titles, posters, ratings, and popularity.

---

##  Tech Stack

* HTML
* CSS
* JavaScript (Vanilla JS)

---

##  How It Works

1. Fetch trending movies from TMDB API
2. Store data in JavaScript
3. Render movie cards dynamically
4. Apply search, filter, and sort using array methods (`map`, `filter`, `sort`)
5. Update UI based on user interaction
